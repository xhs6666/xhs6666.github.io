    # Aurora主题详细使用教程

    该文章会从零开始教你使用Vuepress搭建自己的博客网站,如果你会git和node的使用，那就更容易了

    从node，Git安装，到创建vuepress站点

    如果此篇文章对你有帮助，那么可以帮我分享给更多的人么

    # node安装

    # 下载node

    点击下载

    点击上方链接，下载node，或者你也可以到官方网站进行下载

    Download | Node.js (nodejs.org)

    当你下载完成之后，你会得到如下文件

    # 安装过程化

    1. 双击点击安装

    2. 选择一个安装目录

      这里的安装目录可以随便选择，但是一定要记得此路径，在后面需要使用到

    3. 点击next

      WARNING

      这里一定要选择Add to PATH，一定要记得选择Add to PATH，一定要记得选择Add to PATH

      这里还是点击next

    ​ 然后等待安装完成之后，就可以了

    # 设置环境和变量

    TIP

    因为之前选择了App to PATH，所以我们值需要简单设置一下就行

    # 验证

    上一步安装完成之后，按住win r键，输入cmd

    在面板中，输入node -v验证是否安装成功

    当出现v14.17.3时，表示node已经安装成功了，此值也有可能不同，取决于你下的node版本

    # 设置

    1. 进入上一步安装node的目录中，你将看到如下文件

    2. 在此目录中，新建node_cache和node_global两个文件夹

    3. 在cmd面板中，运行下面两个命令，一个一个的运行

      npm config set prefix "D:\Develop\nodejs\node_global"
npm config set cache "D:\Develop\nodejs\node_cache"
      1
      2

      TIP

      将后面的路径改成你自己对应文件夹就行

    4. 然后再运行下面这个命令

      npm config set registry=http://registry.npm.taobao.org/
      1

      使用淘宝镜像，不然下载速度非常慢

    5. 设置环境变量

    设置好之后，点击确定就行了

    # 验证

    现在验证一下，刚才安装的是否正确

    打开cmd面板，运行下面命令

    npm root-g
    1

    如果输入命令之后，输出的node_modules前面的路径就是你刚才创建的node_global的路径，那么就表示你设置成功

    # 验证脚手架

    TIP

    一定要先验证aurora命令是否可用，否则安装主题会有点麻烦

    在cmd中输入下面命令安装Aurora脚手架

    npm i vuepress-theme-cli -g
    1

    待下载成功之后，在控制台中输入aurora blog，然后按回车

    如果提示下图

    WARNING

    'aurora' 不是内部或外部命令，也不是可运行的程序
或批处理文件。
    1
    2

    进入到环境变量中，新建一个，输入创建的node_global路径

    TIP

    正常情况，你输入aurora blog之后，便会下载一个模板

    然后现在你可以了

    根据面板中的指示，运行相应命令，就可以启动博客了
